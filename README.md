# Customer Sales Analysis

Excel-based analysis of customer sales transactions, covering data cleaning, revenue breakdowns, regional performance, customer segmentation, and a KPI dashboard.

## 📂 File
- `Customer_Sales_Analysis.xlsx`

## 📊 What's Inside
| **Sheet** | **Description** |
| **Raw Sales Data** | Original transaction-level data (20,000 rows) — Transaction ID, Date, Customer, Region, Category, Product, Revenue, Profit, Payment Mode, etc.|
| **Cleaned Sales Data** | Processed version of the raw data with a derived `Month-Year` column for time-based analysis.|
| **Revenue by Category** | Aggregated revenue, profit, profit margin, and revenue share per product category.|
| **Monthly Revenue Trend** | Month-over-month revenue, profit, average order value, and growth rate.|
| **Regional Analysis** | Order volume, revenue, profit, and margin broken down by region.|
| **Customer Segmentation** | Per-customer order count, total revenue, average order value, recency, and segment classification (e.g. Loyal Customer).|
| **KPI Dashboard** | Executive summary view with key metrics, pivot tables, and charts.|

## 🔑 Key Fields
Transaction ID, Date, Customer ID, Customer Name, Age, Gender, Region, Category, Sub-Category, Product, Quantity, Unit Price, Discount %, Revenue, Profit, Payment Mode

## 🛠 Tools Used
- Microsoft Excel (formulas, summary tables, charts/dashboard)

## 📌 How to Use
1. Download `Customer_Sales_Analysis.xlsx`
2. Open in Excel (or compatible software, e.g. Google Sheets)
3. Start with the **KPI Dashboard** sheet for a quick summary, or explore individual analysis sheets for details
